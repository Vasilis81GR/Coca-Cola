Ρυθμίσεις

Cloud backup (owner)

Μόνο για τον ιδιοκτήτη. Βάλε το master token — οι επαφές σου θα σώζονται κρυπτογραφημένα στον server σου, ώστε να τις ανακτάς σε νέα συσκευή.

Cloud επαναφορά (owner)

Μόνο ο ιδιοκτήτης. Βάλε το master token και τον κωδικό σου.

Όρισε κωδικό

Κλειδώνει την ταυτότητά σου. Κράτα και ένα backup — αν χαθεί η συσκευή, με το backup + τον κωδικό ξαναμπαίνεις.
